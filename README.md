# Python Keylogger
  A keylogger is a program that captures keyboard inputs (keystrokes). It can be used for legal purposes like parental control or personal productivity tracking â€” not for hacking or spying.

#  what It Does
This key logger:
  - Captures every keystroke using pynput
  - Saves the keystrokes to a file named keylog.txt

# Requirements
  - Python 3.X
  - pynput
Install with :
  pip install pynput
Run the code :
    python keylogger.py
